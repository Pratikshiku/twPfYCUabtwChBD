## Tourism Information Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3c5e66fce9fa4a629eb22345a6c179ce/ghb20250917) 


### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`/`MybatisPlus`/`jdk1.8`/.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`/`ElementUI`, etc.
- Database: `MySQL`/`MybatisPlus`/

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular tourists, administrators.
Main functions: Frontend (for users, administrators can also use), Background management (only for administrators)

**Frontend (for users, administrators can also use) Functions:**
1. Login
2. Register
3. Display Tourism News
4. Scenic Area Information
5. Food Information
6. Tourism Route Search
7. Online Messages
8. Scenic Spot Details
9. Tourism News

**Background Management (only for administrators) Functions:**
1. Login
2. Account Management (add users, set administrators, password modification)
3. Region Management (add regions, query)
4. Scenic Spot Information Management
5. Local Food Management
6. Tourism Route Management
7. Order Information Management
8. News Management
9. System Management (carousel images, messages, links)
